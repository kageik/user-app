### ディレクトリ構成

```
src
├─app # ルーティング
├─assets
├─components # アプリケーション全体で使用する共通コンポーネント
│  ├─elements # 部品（ボタンなど）
│  └─layouts
├─config
├─constants
├─features # 機能ベースモジュール
├─hooks
├─layouts # ページごとのレイアウト
├─stories
│  └─assets
├─types
└─utils
```

### ディレクトリ・ファイル命名規則

| 対象ディレクトリ・ファイル | 命名規則                           |
| -------------------------- | ---------------------------------- |
| components                 | パスカルケース<br />（PascalCase） |
| hooks                      | キャメルケース<br />（camelCase）  |
| その他                     | ケバブケース<br />（kebab-case）   |
